What is Amaltools?

Amaltools is a website containing a suite of tools to help in a range of situations. The tools are made to make it easier to complete tasks, calculations and more. There are math tools, file tools and a many others!

Who is AMALtools for?

Amaltools is for anyone who needs to do a simple task that they don't want to do manually. For example, if you have a list of numbers that you want to sort in ascending order, you can use the 'Sort Numbers' tool to do it for you.

What does 'AMALtools' mean?

Amaltools is a combination of the words 'AMAL' and 'Tools'. Amal is my name and tools is what the website provides, a suite of (hopefully) useful tools that can help the everyday internet user get simple tasks done. I chose this name because it is unique and it includes my name.

Why did you make Amaltools?

I made Amaltools because I wanted to make something that would be useful to people. I also wanted to make something that would be a challenge to create and would help me learn skills and gain new knowledge. A lot of the tools are actually catered to my needs (getting user feedback has been difficult) as I often find myself like converting an image into another format that are hard to perform on other websites.

How did you make Amaltools?

I made Amaltools using Django, a Python web framework. One of the reasons I chose Django because python was the only programming language I knew at the inception of this project not only that, it is also a very popular framework and has a lot of support and documentation with lots of big companies such as Instagram and Spotify using it. For the frontend I used bootstrap v5.3. Many tools are built using open source libraries which are listed in the acknoledgments page.
